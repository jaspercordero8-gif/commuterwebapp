# commuterwebapp
version control for software development project 

Urbannav  - a smart commuter web app that helps users find routes using real time transport

problem&purpose:
- commuters often need multiple apps for routes,weather,transport updates
- this project combines everything in one platform

Features:
- route planning (walking , driving , public transport,cycling)
- real-time weather & transpor integration
- saved favourite routes
- estimation calories burned(walking , cycling)
- eco-impact indicator
- weekly welness dashboard

Tech stack:
- frontend : HTML ,CSS ,JAVASCRIPT
- Backend: PHP
- Database:MYSQL
- APIS: Google Maps , weather , transport
- tools : MAMP , Github

Installation & setup
1. clone the respository (final versions of codes have "final" in the name)
2. open MAMP (other localhost server)
3. place project in local server folder (if using mamp in htdocs)
4. type in local browser e.g chrome "http://localhost:8888/urbannav/home.html" (this will direct you to the fronpage of the webapp)

usage :
1. enter starting location and destination
2. tell system whats your enerygy (low/high) to determine routes
3. enter departure time
4. press GO
